# AuraStand

AuraStand is a landscape standby dashboard inspired by modern standby displays and built using Kotlin and Jetpack Compose.

## Features

* Minimal & Flip Clock Themes
* AMOLED Black Mode
* Infinite Swipe Navigation
* Auto Rotate Widgets
* Custom Wallpapers
* Photo Slideshow Widget
* Calendar Widget
* Thought Widget
* Battery Widget
* OLED Burn-In Protection
* Fullscreen Landscape Experience
